波動光學公式

駐波、折射、干涉

波動基本式

v = fλ
波速 = 頻率 × 波長

駐波

弦或管中形成駐波時，長度與波長的關係：

L = n · λ/2
兩端固定（或兩端開放）
L = (2n−1) · λ/4
一端固定、一端開放
n = 1, 2, 3... 為正整數，代表諧波數

折射

n = c / v
折射率 = 真空光速 / 介質中光速
n₁ sin θ₁ = n₂ sin θ₂
司乃耳定律（Snell's Law）
n₁ / n₂ = v₂ / v₁ = λ₂ / λ₁
折射率與速度、波長成反比

雙狹縫干涉

δ = d sin θ ≈ dy / L
光程差（d：狹縫間距，L：屏幕距離，y：離中央距離）
亮紋：δ = nλ
n = 0, 1, 2, 3...
暗紋：δ = (2n+1) · λ/2
n = 0, 1, 2, 3...
Δy = Lλ / d
相鄰亮紋（或暗紋）間距
yn = n · Lλ / d
第 n 條亮紋位置（中央為 n=0）

📝 快速記憶

  • 駐波：兩端同 → λ/2；一開一閉 → λ/4（乘奇數）
  • 干涉亮暗：亮 = 整數λ，暗 = 半整數λ
  • 條紋間距：Δy = Lλ/d（L↑或λ↑則變寬，d↑則變窄）
