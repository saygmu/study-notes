合理性檢定解題技巧

Reasonableness Check · 數學解題的最後防線

什麼是合理性檢定？

合理性檢定是在解完題目之後，快速驗證答案是否「合理」的方法。它不是嚴格的證明，而是一種直覺性的檢查——幫助你在考試中抓出明顯的計算錯誤，避免不必要的失分。

💡 核心觀念：合理性檢定的目標不是「重新算一遍」，而是用不同的角度快速確認答案有沒有問題。花 10 秒檢查，可能救回 5 分。

技巧一：代入特殊值

把簡單的數字代入原式，看看答案是否成立。常用的特殊值包括 0、1、−1。

例題 1：因式分解驗證

將 x² + 5x + 6 因式分解。

你的答案：(x + 2)(x + 3)

合理性檢定：代入 x = 1

左式：1 + 5 + 6 = 12
右式：(1+2)(1+3) = 3 × 4 = 12 ✓

再代入 x = 0
左式：0 + 0 + 6 = 6
右式：(0+2)(0+3) = 2 × 3 = 6 ✓
兩個特殊值都吻合 → 答案很可能正確
例題 2：方程式求解驗證

解方程式 2x − 7 = 3。

你的答案：x = 5

合理性檢定：代回原式

左式：2(5) − 7 = 10 − 7 = 3
右式：3 ✓
直接代回原式是最基本也最有效的檢定方式

技巧二：估算數量級

不用精確計算，只要粗略估計答案的大小，看看有沒有差太多。

例題 3：估算驗證

計算 48 × 52。

你的答案：2496

合理性檢定：估算

48 × 52 ≈ 50 × 50 = 2500

更精確地：利用平方差 (50−2)(50+2) = 50² − 4 = 2496 ✓
估算值 2500 與答案 2496 非常接近 → 合理
例題 4：根號估算

計算 √50 的近似值。

你的答案：7.07

合理性檢定：

7² = 49，8² = 64
所以 √50 應在 7 與 8 之間，且非常靠近 7

7.07² = 49.98 ≈ 50 ✓
答案落在合理範圍內 → 正確

技巧三：檢查邊界與極端情況

把 x 推到極端值（趨近 0、趨近無窮大、趨近邊界），看看函數行為是否合理。

例題 5：函數行為檢查

求 f(x) = (x² − 1)/(x − 1) 在 x ≠ 1 時的化簡結果。

你的答案：f(x) = x + 1

合理性檢定：代入極端值

當 x = 100：
原式 = (10000 − 1)/(100 − 1) = 9999/99 ≈ 101
化簡式 = 100 + 1 = 101 ✓

當 x = 0：
原式 = (0 − 1)/(0 − 1) = 1
化簡式 = 0 + 1 = 1 ✓
極端值與普通值都一致 → 化簡正確

技巧四：維度與單位分析

在應用題中，檢查答案的單位是否正確，數量級是否合理。

例題 6：應用題單位檢查

一輛車以時速 60 公里行駛 2.5 小時，求行駛距離。

你的答案：150 公里

合理性檢定：

① 單位分析：(公里/小時) × 小時 = 公里 ✓
② 數量級：60 × 2.5 → 60 × 2 = 120，60 × 3 = 180
　 150 在 120 和 180 之間 ✓
③ 生活經驗：開車 2.5 小時走 150 公里，大概是台北到新竹的距離 → 合理 ✓
單位正確、數量級合理、符合生活經驗 → 答案可信

技巧五：奇偶性與正負號

快速判斷答案的正負號和奇偶性是否合理。

例題 7：正負號判斷

計算 (−3)⁴ × (−2)³。

你的答案：−648

合理性檢定：

(−3)⁴：偶數次方 → 正數 → 81
(−2)³：奇數次方 → 負數 → −8
正 × 負 = 負 ✓

數量級：81 × 8 = 648 ✓
符號正確、數值正確 → 答案為 −648

技巧六：圖形直覺

在腦中（或草稿紙上）快速畫出大致的圖形，看看答案是否與圖形一致。

例題 8：交點問題

求 y = x² 與 y = 2x + 3 的交點。

你的答案：(3, 9) 和 (−1, 1)

合理性檢定：

① 代入 (3, 9)：
　 拋物線：3² = 9 ✓
　 直線：2(3)+3 = 9 ✓

② 代入 (−1, 1)：
　 拋物線：(−1)² = 1 ✓
　 直線：2(−1)+3 = 1 ✓

③ 圖形直覺：拋物線開口向上，直線斜率為正且截距為 3，應該有兩個交點，一個在右上方、一個在左邊 → 符合 ✓
代入驗證 + 圖形直覺都吻合 → 答案正確

技巧七：反向運算

用逆運算來檢驗：做完乘法就用除法驗算、解完微分就積分回去。

例題 9：除法反驗

計算 372 ÷ 12。

你的答案：31

合理性檢定：反向運算

31 × 12 = 31 × 10 + 31 × 2 = 310 + 62 = 372 ✓
逆運算結果一致 → 答案正確

常見錯誤類型速查

錯誤類型 快速檢查法
正負號算錯 只追蹤符號，不管數值
小數點位移 估算數量級是否合理
因式分解錯誤 代入 x = 0, 1, −1 驗證
方程式解錯 代回原式左右比較
化簡出錯 代入大數或特殊值比較
單位搞混 維度分析：追蹤單位的運算
漏解或增根 每個解都代回去，檢查定義域

綜合練習

練習 1

小明算出 √(144 + 25) = √144 + √25 = 12 + 5 = 17。請用合理性檢定找出他的錯誤。

合理性檢定：

√(144 + 25) = √169 = 13

小明的錯誤：根號不能分配到加法上！
√(a + b) ≠ √a + √b

快速檢查：如果 √(a+b) = √a + √b，那代入 a = b = 1：
左式 = √2 ≈ 1.41，右式 = 1 + 1 = 2 → 明顯不等 ✗

正確答案：13
練習 2

解方程式 x² − 5x + 6 = 0，某同學得到 x = 1 和 x = 6。用合理性檢定判斷是否正確。

合理性檢定 1：代入驗證

代入 x = 1：1 − 5 + 6 = 2 ≠ 0 ✗ → 馬上知道不對！

合理性檢定 2：韋達定理

兩根之和 = −(−5)/1 = 5
同學的答案：1 + 6 = 7 ≠ 5 ✗

兩根之積 = 6/1 = 6
同學的答案：1 × 6 = 6 ✓ （積湊巧對了，但和不對）

正確答案：x = 2 和 x = 3
驗證：2 + 3 = 5 ✓，2 × 3 = 6 ✓
練習 3

計算 log₁₀(500) 的值。某同學得到 3.7。請判斷是否合理。

合理性檢定：邊界估算

log₁₀(100) = 2
log₁₀(1000) = 3

因為 100 < 500 < 1000
所以 2 < log₁₀(500) < 3

但同學的答案 3.7 > 3 → 不合理！

正確答案：log₁₀(500) ≈ 2.699

考試實戰建議

合理性檢定三步口訣

算完先停一下想，
代個數字看看像不像，
符號單位別搞忘！

⏱️ 時間分配建議：選擇題每題留 10～15 秒 做合理性檢定。計算題在交卷前統一用 5 分鐘 掃一遍關鍵步驟。養成習慣後，很多檢查幾乎是自動完成的。
🎯 最高效的檢查優先順序：
① 代回原式（最直接）→ ② 正負號與奇偶性（最快）→ ③ 估算數量級（最實用）→ ④ 特殊值代入（最萬用）
