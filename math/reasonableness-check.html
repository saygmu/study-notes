合理性檢定解題技巧

Hypothesis Testing · 統計推論的核心工具

什麼是合理性檢定？

合理性檢定（假設檢定）是統計推論的方法之一。核心想法是：先假設某個命題成立（虛無假設 H₀），然後用抽樣資料來判斷這個假設是否「不合理」。如果資料所呈現的結果在 H₀ 之下非常不可能發生，就有理由拒絕 H₀。

💡 直覺理解：就像法庭上的「無罪推定」——先假設被告無罪（H₀），除非證據強到「超越合理懷疑」，才判定有罪（拒絕 H₀）。

基本架構

H₀：虛無假設（Null Hypothesis）
想要推翻的假設，通常是「沒有差異」「沒有效果」「符合原本的分布」
H₁：對立假設（Alternative Hypothesis）
研究者想要支持的假設，與 H₀ 互斥
名詞 意義
顯著水準 α 拒絕 H₀ 所容許的最大犯錯機率，常取 0.05 或 0.01
檢定統計量 從樣本資料計算出來的數值，用來衡量資料偏離 H₀ 的程度
拒絕域 當檢定統計量落在此範圍時，拒絕 H₀
p 值 在 H₀ 為真的前提下，觀察到「目前資料或更極端結果」的機率
型一錯誤（α 錯誤） H₀ 為真，卻拒絕了 H₀（冤枉好人）
型二錯誤（β 錯誤） H₀ 為假，卻沒有拒絕 H₀（放走壞人）

解題五步驟

標準流程
Step 1：建立假設
寫出 H₀ 和 H₁，確定是單尾還是雙尾檢定。
Step 2：決定顯著水準 α
題目通常會給（如 α = 0.05），沒給就用 0.05。
Step 3：計算檢定統計量
根據題型選用適當公式（Z 檢定、t 檢定、χ² 檢定等）。
Step 4：做出決策
比較檢定統計量與臨界值（或比較 p 值與 α）。
· 若 |統計量| > 臨界值 → 拒絕 H₀
· 若 p 值 < α → 拒絕 H₀
Step 5：寫結論
用日常語言解釋結果。「在顯著水準 α = 0.05 下，有充分證據拒絕 H₀，即⋯⋯」

母體比例的 Z 檢定

適用於：檢定母體比例 p 是否等於（或大於/小於）某個值 p₀。

Z = (p̂ − p₀) / √(p₀(1−p₀)/n)
p̂ = 樣本比例，p₀ = H₀ 假設的比例，n = 樣本數
例題 1：硬幣是否公正？

擲一枚硬幣 200 次，出現正面 118 次。在 α = 0.05 下，能否認為這枚硬幣不公正？

Step 1：H₀：p = 0.5（硬幣公正），H₁：p ≠ 0.5（雙尾）

Step 2：α = 0.05，雙尾臨界值 Z* = ±1.96

Step 3：
p̂ = 118/200 = 0.59
Z = (0.59 − 0.5) / √(0.5×0.5/200)
= 0.09 / √(0.00125)
= 0.09 / 0.03536
≈ 2.546

Step 4：|Z| = 2.546 > 1.96 → 落在拒絕域
結論：在 α = 0.05 下，拒絕 H₀，有充分證據認為這枚硬幣不公正。

母體平均數的檢定

已知母體標準差 σ 時用 Z 檢定，未知 σ 時用 t 檢定。

Z = (X̄ − μ₀) / (σ/√n)
已知 σ 時使用
t = (X̄ − μ₀) / (s/√n)
未知 σ 時使用，自由度 df = n − 1
例題 2：燈泡壽命是否達標？

某廠商宣稱燈泡平均壽命 1000 小時（σ = 50）。隨機抽 36 個，平均壽命 985 小時。在 α = 0.05 下，是否有理由懷疑燈泡壽命未達標？

Step 1：H₀：μ = 1000，H₁：μ < 1000（左尾）

Step 2：α = 0.05，左尾臨界值 Z* = −1.645

Step 3：
Z = (985 − 1000) / (50/√36)
= −15 / (50/6)
= −15 / 8.333
≈ −1.80

Step 4：Z = −1.80 < −1.645 → 落在拒絕域
結論：在 α = 0.05 下，拒絕 H₀，有充分證據認為燈泡平均壽命低於 1000 小時。

卡方適合度檢定（χ² Goodness-of-Fit）

用於檢定觀察到的次數分布是否符合期望的理論分布。

χ² = Σ (Oᵢ − Eᵢ)² / Eᵢ
Oᵢ = 第 i 類的觀察次數，Eᵢ = 第 i 類的期望次數，df = k − 1
例題 3：骰子是否公正？

擲骰子 120 次，各點數出現次數如下。在 α = 0.05 下，骰子是否公正？

點數123456
觀察次數 O251715232416
期望次數 E202020202020
Step 1：H₀：骰子公正（各點數機率 = 1/6），H₁：骰子不公正

Step 2：α = 0.05，df = 6 − 1 = 5，臨界值 χ²* = 11.07

Step 3：
χ² = (25−20)²/20 + (17−20)²/20 + (15−20)²/20 + (23−20)²/20 + (24−20)²/20 + (16−20)²/20
= 25/20 + 9/20 + 25/20 + 9/20 + 16/20 + 16/20
= 1.25 + 0.45 + 1.25 + 0.45 + 0.80 + 0.80
= 5.00

Step 4：χ² = 5.00 < 11.07 → 沒有落在拒絕域
結論：在 α = 0.05 下，不拒絕 H₀，沒有充分證據認為骰子不公正。
⚠️ 注意：「不拒絕 H₀」不等於「接受 H₀」！只是目前的證據不足以推翻它而已。

卡方獨立性檢定

用於檢定兩個類別變數之間是否獨立（有無關聯）。

Eᵢⱼ = (列合計 × 行合計) / 總合計
期望次數的計算方式，df = (r−1)(c−1)
例題 4：性別與選課偏好是否獨立？

調查 200 名學生的性別與選修科目，在 α = 0.05 下檢定兩者是否獨立。

文科理科合計
男生3070100
女生5050100
合計80120200
Step 1：H₀：性別與選課偏好獨立，H₁：兩者有關聯

Step 2：α = 0.05，df = (2−1)(2−1) = 1，臨界值 χ²* = 3.841

Step 3：先算期望次數
E₁₁ = 100×80/200 = 40，E₁₂ = 100×120/200 = 60
E₂₁ = 100×80/200 = 40，E₂₂ = 100×120/200 = 60

χ² = (30−40)²/40 + (70−60)²/60 + (50−40)²/40 + (50−60)²/60
= 100/40 + 100/60 + 100/40 + 100/60
= 2.5 + 1.667 + 2.5 + 1.667
≈ 8.334

Step 4：χ² = 8.334 > 3.841 → 落在拒絕域
結論：在 α = 0.05 下，拒絕 H₀，有充分證據認為性別與選課偏好之間存在關聯。

解題技巧與陷阱

技巧 1：判斷單尾 vs 雙尾

看題目關鍵字：

· 「是否不同」「是否改變」「是否不等於」→ 雙尾（H₁：μ ≠ μ₀）
· 「是否大於」「是否提高」「是否增加」→ 右尾（H₁：μ > μ₀）
· 「是否小於」「是否降低」「是否減少」→ 左尾（H₁：μ < μ₀）
技巧 2：p 值法 vs 臨界值法
臨界值法：比較統計量和臨界值的大小
→ 適合考試，臨界值通常會給

p 值法：比較 p 值和 α 的大小
→ 若 p < α，拒絕 H₀
→ p 值越小，拒絕 H₀ 的理由越充分
技巧 3：χ² 檢定的前提條件
· 每一格的期望次數 E ≥ 5（不是觀察次數！）
· 若有 E < 5 的格子，需合併類別
· χ² 值永遠 ≥ 0，且 χ² 檢定都是右尾檢定
⚠️ 常見陷阱：
① 拒絕域搞反：χ² 檢定永遠看右尾，不會有「左尾拒絕」的情況
② 混淆 O 和 E：公式是 (O−E)²/E，分母是 E 不是 O
③ 結論用詞不當：只能說「拒絕 H₀」或「不拒絕 H₀」，不能說「接受 H₀」
④ 忘記算自由度：適合度 df = k−1，獨立性 df = (r−1)(c−1)

綜合練習

練習 1

某校聲稱其畢業生起薪平均為 35000 元。隨機抽樣 25 名畢業生，得平均薪資 33800 元，樣本標準差 3000 元。在 α = 0.05 下，檢定該校的聲稱是否合理。

Step 1：H₀：μ = 35000，H₁：μ ≠ 35000（雙尾）

Step 2：未知 σ，用 t 檢定，df = 24，α = 0.05，臨界值 t* ≈ ±2.064

Step 3：
t = (33800 − 35000) / (3000/√25)
= −1200 / 600 = −2.00

Step 4：|t| = 2.00 < 2.064 → 沒有落在拒絕域

結論：在 α = 0.05 下，不拒絕 H₀，沒有充分證據否定起薪平均 35000 元的聲稱。
練習 2

某商店記錄一週各天的顧客數如下，在 α = 0.05 下檢定顧客數是否平均分布。

星期一二三四五六日
顧客數50453842567267
Step 1：H₀：顧客數平均分布（每天機率 = 1/7），H₁：不是平均分布

Step 2：α = 0.05，df = 7−1 = 6，臨界值 χ²* = 12.59

Step 3：
總顧客數 = 50+45+38+42+56+72+67 = 370
每天期望 E = 370/7 ≈ 52.86

χ² = (50−52.86)²/52.86 + (45−52.86)²/52.86 + (38−52.86)²/52.86 + (42−52.86)²/52.86 + (56−52.86)²/52.86 + (72−52.86)²/52.86 + (67−52.86)²/52.86

= 0.155 + 1.169 + 4.176 + 2.230 + 0.187 + 6.935 + 3.784
≈ 18.636

Step 4：χ² = 18.636 > 12.59 → 落在拒絕域

結論：在 α = 0.05 下，拒絕 H₀，有充分證據認為各天的顧客數並非平均分布。（週末明顯較多）
練習 3

某藥廠聲稱新藥的有效率為 80%。臨床試驗中 100 人有 72 人有效。在 α = 0.05 下，能否認為有效率低於 80%？

Step 1：H₀：p = 0.80，H₁：p < 0.80（左尾）

Step 2：α = 0.05，左尾臨界值 Z* = −1.645

Step 3：
p̂ = 72/100 = 0.72
Z = (0.72 − 0.80) / √(0.80×0.20/100)
= −0.08 / √(0.0016)
= −0.08 / 0.04 = −2.00

Step 4：Z = −2.00 < −1.645 → 落在拒絕域

結論：在 α = 0.05 下，拒絕 H₀，有充分證據認為新藥的有效率低於 80%。
