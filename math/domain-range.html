函數的定義域與值域

Domain and Range

基本概念

定義域 (Domain)
函數 f(x) 中，所有可代入的 x 值所形成的集合
值域 (Range)
函數 f(x) 中，所有可能輸出的 y 值所形成的集合

求定義域的常見限制

函數類型 限制條件 例子
分式 分母 ≠ 0 1/(x−2) → x ≠ 2
偶次根號 被開方數 ≥ 0 √(x−3) → x ≥ 3
對數 真數 > 0 log(x+1) → x > −1
指數 無限制 2ˣ → x ∈ ℝ
多項式 無限制 x² + 3x → x ∈ ℝ
⚠️ 注意：遇到複合函數，要同時滿足所有限制條件！

定義域例題

例題 1：分式函數

f(x) = (x+1) / (x²−4)

解：分母不能為 0
x² − 4 ≠ 0
(x+2)(x−2) ≠ 0
x ≠ 2 且 x ≠ −2
定義域：{x ∈ ℝ | x ≠ ±2}
例題 2：根號函數

f(x) = √(2x−6)

解：被開方數 ≥ 0
2x − 6 ≥ 0
x ≥ 3
定義域：[3, +∞)
例題 3：對數函數

f(x) = log₂(x²−x−6)

解：真數 > 0
x² − x − 6 > 0
(x−3)(x+2) > 0
x < −2 或 x > 3
定義域：(−∞, −2) ∪ (3, +∞)
例題 4：複合限制

f(x) = √(x+4) / (x−1)

解：需同時滿足：
① 根號內 ≥ 0：x + 4 ≥ 0 → x ≥ −4
② 分母 ≠ 0：x ≠ 1
定義域：[−4, 1) ∪ (1, +∞)
例題 5：根號套根號

f(x) = √(4 − √x)

解：由內而外：
① 內層：x ≥ 0
② 外層：4 − √x ≥ 0 → √x ≤ 4 → x ≤ 16
定義域：[0, 16]

求值域的方法

方法 1：配方法（二次函數）

f(x) = x² − 4x + 7

解：配方
= (x−2)² + 3
頂點為 (2, 3)，開口向上
值域：[3, +∞)
方法 2：判別式法

y = (x²+1) / (x²−1)（x ≠ ±1）

解：整理成 x 的方程式
y(x²−1) = x² + 1
yx² − y = x² + 1
x²(y−1) = y + 1
x² = (y+1)/(y−1)

因為 x² ≥ 0，所以 (y+1)/(y−1) ≥ 0
解不等式：y ≤ −1 或 y > 1
值域：(−∞, −1] ∪ (1, +∞)
方法 3：單調性分析

f(x) = √(x−2) + 3，x ≥ 2

解：
· √(x−2) 隨 x 增大而增大（遞增）
· 當 x = 2 時，f(2) = 0 + 3 = 3（最小值）
· 當 x → +∞ 時，f(x) → +∞
值域：[3, +∞)
方法 4：換元法

f(x) = x + √(1−x)

解：令 t = √(1−x)，則 t ≥ 0，且 x = 1 − t²

f = (1−t²) + t = −t² + t + 1
= −(t − ½)² + 5/4

· t ≥ 0 時，最大值在 t = ½ 處，f_max = 5/4
· 當 t → +∞，f → −∞
值域：(−∞, 5/4]
方法 5：分離常數法

f(x) = (2x−1)/(x+1)，x > 0

解：改寫
f(x) = [2(x+1)−3]/(x+1) = 2 − 3/(x+1)

當 x > 0：
· x + 1 > 1
· 0 < 3/(x+1) < 3
· −3 < −3/(x+1) < 0
· −1 < 2 − 3/(x+1) < 2
值域：(−1, 2)

綜合練習

練習 1

求 f(x) = √(x−1) + √(5−x) 的定義域與值域。

定義域：[1, 5]

值域：
令 y = √(x−1) + √(5−x)
y² = (x−1) + 2√[(x−1)(5−x)] + (5−x) = 4 + 2√[(x−1)(5−x)]

設 t = (x−1)(5−x) = −x²+6x−5 = −(x−3)² + 4
當 x ∈ [1,5] 時，t ∈ [0, 4]

所以 y² ∈ [4, 8]
y ∈ [2, 2√2]
練習 2

求 f(x) = (x²+2x+3)/(x²+1) 的值域。

用判別式法：

y(x²+1) = x²+2x+3
(y−1)x² − 2x + (y−3) = 0

情況 1：y = 1 時，−2x − 2 = 0，x = −1（有解）

情況 2：y ≠ 1 時，需 Δ ≥ 0
4 − 4(y−1)(y−3) ≥ 0
1 − (y²−4y+3) ≥ 0
y² − 4y + 2 ≤ 0
y = (4 ± √8)/2 = 2 ± √2
2−√2 ≤ y ≤ 2+√2

值域：[2−√2, 2+√2]
