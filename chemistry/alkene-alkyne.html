烯類與炔類

有機化學 · 反應與性質整理

一、製備方法

烯類

石油裂煉（工業法）：大分子烷烴高溫斷裂。

$$\text{CH}_3\text{CH}_2\text{CH}_3 \xrightarrow{\Delta} \text{CH}_3\text{CH}{=}\text{CH}_2 + \text{H}_2$$

醇類脫水（實驗室法）：加入濃 $\text{H}_2\text{SO}_4$ 加熱至約 180°C，發生分子內脫水。

$$\text{CH}_3\text{CH}_2\text{OH} \xrightarrow[\sim 180°\text{C}]{\text{conc. H}_2\text{SO}_4} \text{CH}_2{=}\text{CH}_2 + \text{H}_2\text{O}$$

鹵代烷脫去鹵化氫（實驗室法）：鹵代烷與 KOH 的乙醇溶液共熱。

$$\text{CH}_3\text{CH}_2\text{Br} + \text{KOH}(\text{alc}) \xrightarrow{\Delta} \text{CH}_2{=}\text{CH}_2 + \text{KBr} + \text{H}_2\text{O}$$

炔類（以乙炔為例）

電石（碳化鈣 $\text{CaC}_2$）加水：

$$\text{(1)}\ \text{CaCO}_3 \xrightarrow{\Delta} \text{CaO} + \text{CO}_2$$ $$\text{(2)}\ \text{CaO} + 3\text{C} \xrightarrow{3000°\text{C}} \text{CaC}_2 + \text{CO}$$ $$\text{(3)}\ \text{CaC}_2 + 2\text{H}_2\text{O} \rightarrow \text{HC}{\equiv}\text{CH} + \text{Ca(OH)}_2$$

四、末端炔專屬 — 金屬取代反應

末端炔 ($\text{R-C}{\equiv}\text{C-H}$) 的氫具有微酸性，可被金屬取代。

加硝酸銀氨溶液（妥倫試劑）：

$$\text{HC}{\equiv}\text{CH} + 2\text{Ag(NH}_3\text{)}_2\text{NO}_3 \rightarrow \text{AgC}{\equiv}\text{CAg}{\downarrow}\;(\text{白色沉澱}) + 2\text{NH}_4\text{NO}_3 + 2\text{NH}_3$$
白色沉澱為乙炔銀

加氯化亞銅氨溶液：

$$\text{HC}{\equiv}\text{CH} + 2\text{Cu(NH}_3\text{)}_2\text{Cl} \rightarrow \text{CuC}{\equiv}\text{CCu}{\downarrow}\;(\text{紅色沉澱}) + 2\text{NH}_4\text{Cl} + 2\text{NH}_3$$
紅色沉澱為乙炔亞銅
考試重點

乙炔銀（白色）與乙炔亞銅（紅色）皆為末端炔的專屬檢驗。非末端炔（如 $\text{CH}_3\text{C}{\equiv}\text{CCH}_3$）不會產生沉澱。此外，乙炔銀與乙炔亞銅乾燥後易爆炸，需妥善處理。
