兩性物質

既能與酸反應，也能與鹼反應的物質。

兩性氫氧化物：Al、Zn、Pb、Sn、Cr 的氫氧化物

Al(OH)₃

與酸：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O
與鹼：Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O

Zn(OH)₂

與酸：Zn(OH)₂ + 2H⁺ → Zn²⁺ + 2H₂O
與鹼：Zn(OH)₂ + 2OH⁻ → ZnO₂²⁻ + 2H₂O

鋁的兩性

與酸：2Al + 6HCl → 2AlCl₃ + 3H₂↑
與鹼：2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑

錯合物

由中心金屬離子與配位基以配位鍵結合形成的化合物。

常見配位基

配位基 配位數
NH₃、H₂O、CN⁻、Cl⁻、OH⁻單牙
乙二胺 (en)雙牙
EDTA六牙

重要錯離子

錯離子 顏色 用途
[Ag(NH₃)₂]⁺ 無色 多倫試劑
[Cu(NH₃)₄]²⁺ 深藍 檢驗 Cu²⁺
[Fe(SCN)]²⁺ 血紅 檢驗 Fe³⁺
[Zn(OH)₄]²⁻ 無色 Zn(OH)₂ 溶於過量 NaOH

錯合反應

Cu²⁺ + 4NH₃ → [Cu(NH₃)₄]²⁺（深藍色）
Ag⁺ + 2NH₃ → [Ag(NH₃)₂]⁺（無色）
Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺（血紅色）

綜合應用

分離 Al³⁺ 和 Fe³⁺

加入過量 NaOH → Fe(OH)₃ 沉澱，Al(OH)₃ 溶解成 [Al(OH)₄]⁻ → 過濾分離

分離 Ag⁺ 和 Pb²⁺

加 HCl → 都沉澱為白色 → 加熱水（PbCl₂ 溶解）或加 NH₃（AgCl 溶解成 [Ag(NH₃)₂]⁺）

重點

  • 沉澱：鉀鈉銨硝全溶，鹵化物注意 Ag⁺、Pb²⁺
  • 兩性：Al、Zn、Pb、Sn、Cr 的氫氧化物
  • 錯合：深藍 [Cu(NH₃)₄]²⁺、血紅 [Fe(SCN)]²⁺
  • 分離：利用溶解度差異、兩性、錯合反應
