晶場理論與配位化合物

Crystal Field Theory · 混成軌域與電子組態

一、配位化合物基礎

配位化合物（錯合物）由中心金屬離子與配位基組成。配位基提供孤對電子給金屬離子形成配位鍵。

$[\text{Ni(CN)}_4]^{2-}$
中心金屬：Ni²⁺ ｜ 配位基：CN⁻ ｜ 配位數：4

常見配位基

配位基 提供電子對數 場強
$\text{CN}^-$（氰根） 1 強場
$\text{CO}$（一氧化碳） 1 強場
$\text{NH}_3$（氨） 1 中場
$\text{H}_2\text{O}$（水） 1 弱場
$\text{Cl}^-$（氯離子） 1 弱場
en（乙二胺） 2 中場

二、混成軌域

金屬離子會進行軌域混成，提供空軌域接受配位基的電子對。

混成類型 幾何形狀 配位數 使用軌域
$sp^3$ 四面體 4 1s + 3p
$dsp^2$ 平面四方 4 1d + 1s + 2p
$sp^3d^2$ 或 $d^2sp^3$ 八面體 6 2d + 1s + 3p
💡 內軌與外軌

內軌錯合物：使用 $(n-1)d$ 軌域混成（如 $d^2sp^3$）
外軌錯合物：使用 $nd$ 軌域混成（如 $sp^3d^2$）

三、晶場分裂

當配位基接近金屬離子時，d 軌域會因為靜電排斥而分裂成不同能量。

八面體場

在八面體場中：

  • $e_g$ 軌域（$d_{z^2}, d_{x^2-y^2}$）：指向配位基，能量較高
  • $t_{2g}$ 軌域（$d_{xy}, d_{xz}, d_{yz}$）：指向配位基之間，能量較低

四、強場 vs 弱場

🔵 強場配位基

造成大的分裂能 Δ

  • 電子傾向成對填入低能階
  • 形成低自旋錯合物
  • 例：$\text{CN}^-$, $\text{CO}$

🔴 弱場配位基

造成小的分裂能 Δ

  • 電子傾向先佔據所有軌域
  • 形成高自旋錯合物
  • 例：$\text{Cl}^-$, $\text{H}_2\text{O}$

光譜化學序列

$\text{I}^- < \text{Br}^- < \text{Cl}^- < \text{F}^- < \text{OH}^- < \text{H}_2\text{O} < \text{NH}_3 < \text{en} < \text{NO}_2^- < \text{CN}^- < \text{CO}$
弱場 ←―――――――――――――――――→ 強場

五、高自旋 vs 低自旋

例：$d^5$ 組態（如 Fe³⁺、Mn²⁺）

$t_{2g}$: ↿ ↿ ↿

未成對電子：5 個
具有順磁性

🧲 磁性判斷

順磁性：有未成對電子，會被磁場吸引
反磁性：所有電子都成對，會被磁場排斥

六、平面四方形成條件

平面四方（$dsp^2$ 混成）是一種特殊的四配位幾何構型，電子必須成對才能空出 d 軌域來混成。

📋 形成 $dsp^2$ 的條件（滿足其一即可）

條件 說明 範例
第一列 + 強場 3d 金屬 + 強場配位基 $[\text{Ni(CN)}_4]^{2-}$
第二、三列過渡金屬 4d/5d 金屬本身傾向低自旋 $[\text{PtCl}_4]^{2-}$, $[\text{PdCl}_4]^{2-}$, $[\text{AuCl}_4]^{-}$
$d^8$ 組態 剛好可以填滿 $t_{2g}$ 和部分 $e_g$ Ni²⁺, Pd²⁺, Pt²⁺, Au³⁺

常見離子的混成類型

離子 電子組態 常見混成 幾何形狀
Ni²⁺（$d^8$） $[Ar]3d^8$ $dsp^2$（強場）/ $sp^3$（弱場） 平面四方 / 四面體
Cu²⁺（$d^9$） $[Ar]3d^9$ $dsp^2$ 平面四方
Zn²⁺（$d^{10}$） $[Ar]3d^{10}$ $sp^3$ 四面體
Pt²⁺（$d^8$） $[Xe]4f^{14}5d^8$ $dsp^2$ 平面四方
Pd²⁺（$d^8$） $[Kr]4d^8$ $dsp^2$ 平面四方
Au³⁺（$d^8$） $[Xe]4f^{14}5d^8$ $dsp^2$ 平面四方
Cd²⁺（$d^{10}$） $[Kr]4d^{10}$ $sp^3$ 四面體
Hg²⁺（$d^{10}$） $[Xe]4f^{14}5d^{10}$ $sp^3$ 四面體
💡 $d^{10}$ 為何是 $sp^3$？

$d^{10}$ 組態的 d 軌域全滿，無法空出 d 軌域參與 $dsp^2$ 混成，只能用外層的 s 和 p 軌域形成 $sp^3$。

七、範例解析

例題：$[\text{Ni(CN)}_4]^{2-}$ 的混成軌域

步驟 1：確定中心金屬離子
Ni²⁺ 的電子組態：$[Ar]3d^8$（8 個 d 電子）

步驟 2：判斷配位基場強
CN⁻ 是強場配位基

步驟 3：決定自旋狀態
強場 → 電子成對 → 低自旋

步驟 4：畫出電子排列

3d: [↿⇂][↿⇂][↿⇂][↿⇂][ ]   4s: [ ]   4p: [ ][ ][ ]

空出 1 個 d + 1 個 s + 2 個 p → $dsp^2$ 混成

結論：$dsp^2$ 混成，平面四方形，反磁性（無未成對電子）

例題：$[\text{NiCl}_4]^{2-}$ 的混成軌域

步驟 1：Ni²⁺ → $[Ar]3d^8$

步驟 2：Cl⁻ 是弱場配位基

步驟 3：弱場 → 電子不成對 → 高自旋

步驟 4：畫出電子排列

3d: [↿⇂][↿⇂][↿⇂][↿ ][↿ ]   4s: [ ]   4p: [ ][ ][ ]

3d 軌域沒有空位，使用 4s + 4p → $sp^3$ 混成

結論：$sp^3$ 混成，四面體形，順磁性（有 2 個未成對電子）

八、重點整理

🎯 快速判斷流程

  1. 確定金屬離子的 $d^n$ 組態
  2. 判斷配位基是強場還是弱場
  3. 強場 → 低自旋（電子成對）；弱場 → 高自旋
  4. 根據空軌域決定混成類型

📌 記憶口訣

  • $d^8$ + 強場 或 二三列 → $dsp^2$（平面四方）
  • $d^{10}$（Zn²⁺, Cd²⁺, Hg²⁺）→ $sp^3$（四面體）
  • CN⁻、CO 必強場；Cl⁻、H₂O 必弱場
