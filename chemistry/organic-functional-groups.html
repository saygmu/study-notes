1-5 芳香烴

苯的結構

分子式 $\text{C}_6\text{H}_6$，六個碳形成平面正六邊形，鍵角 120°。碳碳鍵等長（0.139 nm），介於單鍵（0.154）與雙鍵（0.134）之間，屬共振結構（離域 π 電子）。

凱庫勒結構 vs 共振
苯不是「交替的單鍵和雙鍵」，而是六個 π 電子均勻分布在環上。因此苯的化學性質以取代反應為主，而非加成反應（不易破壞穩定的芳香環）。

苯的性質

  • 無色液體，有特殊氣味，有毒（致癌物）
  • 不溶於水，密度小於水（$\rho = 0.879\text{ g/mL}$）
  • 良好的有機溶劑
  • 燃燒時火焰明亮且多煙（含碳比高）

苯的反應

取代反應（主要反應類型）：

$$\text{C}_6\text{H}_6 + \text{Br}_2 \xrightarrow{\text{FeBr}_3} \text{C}_6\text{H}_5\text{Br} + \text{HBr}$$
溴化反應，需催化劑 FeBr₃
$$\text{C}_6\text{H}_6 + \text{HNO}_3 \xrightarrow{\text{H}_2\text{SO}_4} \text{C}_6\text{H}_5\text{NO}_2 + \text{H}_2\text{O}$$
硝化反應，混酸（濃硫酸+濃硝酸）
$$\text{C}_6\text{H}_6 + \text{CH}_3\text{Cl} \xrightarrow{\text{AlCl}_3} \text{C}_6\text{H}_5\text{CH}_3 + \text{HCl}$$
弗瑞乳–乖夫茲烷基化反應（Friedel-Crafts）

加成反應（特殊條件下）：

$$\text{C}_6\text{H}_6 + 3\text{H}_2 \xrightarrow{\text{Ni, 高溫高壓}} \text{C}_6\text{H}_{12}$$
加氫生成環己烷（需高溫高壓 + 催化劑）

常見芳香烴

名稱結構用途
甲苯$\text{C}_6\text{H}_5\text{CH}_3$溶劑、TNT 原料
二甲苯$\text{C}_6\text{H}_4(\text{CH}_3)_2$溶劑、合成纖維原料（對二甲苯→PET）
萘$\text{C}_{10}\text{H}_8$（雙環）樟腦丸、染料中間體
苯乙烯$\text{C}_6\text{H}_5\text{CH}{=}\text{CH}_2$聚苯乙烯（PS）原料
鄰、間、對位
雙取代苯有三種異構物：鄰位（ortho, 1,2）、間位（meta, 1,3）、對位（para, 1,4）。

1-6 官能基與有機鹵化烴

官能基總覽

官能基結構代表類別範例
碳碳雙鍵$\text{C}{=}\text{C}$烯乙烯
碳碳參鍵$\text{C}{\equiv}\text{C}$炔乙炔
鹵素$\text{-X}$（F, Cl, Br, I）鹵化烴氯甲烷
羥基$\text{-OH}$醇、酚乙醇、酚
醚鍵$\text{-O-}$醚乙醚
醛基$\text{-CHO}$醛甲醛
酮基$\text{>C=O}$酮丙酮
羧基$\text{-COOH}$羧酸醋酸
酯基$\text{-COO-}$酯乙酸乙酯
胺基$\text{-NH}_2$胺甲胺
醯胺鍵$\text{-CONH-}$醯胺尿素

有機鹵化烴

烴類中的氫被鹵素取代而成。通式 $\text{R-X}$（X = F, Cl, Br, I）。

  • 製備：烷類 + 鹵素（光照或高溫）→ 鹵化烴 + HX
  • 性質：沸點隨碳數增加而升高，密度通常 > 水

常見鹵化烴：

名稱分子式用途 / 特性
氯仿（三氯甲烷）$\text{CHCl}_3$早期麻醉劑、有機溶劑
四氯化碳$\text{CCl}_4$滅火劑、溶劑（有毒，已限用）
氟氯碳化物（CFCs）如 $\text{CCl}_2\text{F}_2$曾用冷媒，破壞臭氧層
鐵氟龍（PTFE）$(\text{CF}_2\text{CF}_2)_n$不沾鍋塗層、耐化學性
PVC（聚氯乙烯）$(\text{CH}_2\text{CHCl})_n$塑膠管、雨衣
環保議題
CFCs 在平流層經紫外線分解釋出 Cl 自由基，催化分解 $\text{O}_3$，導致臭氧層破洞。《蒙特婁議定書》已全面禁用。

1-7 醇、醚與酚

醇（Alcohol）

通式 $\text{R-OH}$。OH 接在飽和碳上。

分類：

  • 一級醇（1°）：OH 所接的碳連 1 個碳（$\text{RCH}_2\text{OH}$）
  • 二級醇（2°）：OH 所接的碳連 2 個碳（$\text{R}_2\text{CHOH}$）
  • 三級醇（3°）：OH 所接的碳連 3 個碳（$\text{R}_3\text{COH}$）

常見醇類：

名稱分子式特性
甲醇$\text{CH}_3\text{OH}$有毒（失明甚至致死）、木精
乙醇$\text{C}_2\text{H}_5\text{OH}$酒精、消毒（75%）、燃料
乙二醇$\text{HOCH}_2\text{CH}_2\text{OH}$防凍劑（二元醇）
丙三醇（甘油）$\text{C}_3\text{H}_5(\text{OH})_3$保濕劑、炸藥原料（三元醇）

醇的反應：

$$\text{C}_2\text{H}_5\text{OH} \xrightarrow[\text{180°C}]{\text{濃 H}_2\text{SO}_4} \text{CH}_2{=}\text{CH}_2 + \text{H}_2\text{O}$$
分子內脫水 → 烯類（高溫）
$$2\text{C}_2\text{H}_5\text{OH} \xrightarrow[\text{140°C}]{\text{濃 H}_2\text{SO}_4} \text{C}_2\text{H}_5\text{OC}_2\text{H}_5 + \text{H}_2\text{O}$$
分子間脫水 → 醚類（低溫）
$$\text{醇} \xrightarrow{\text{氧化}} \text{1° → 醛 → 羧酸};\quad \text{2° → 酮};\quad \text{3° → 不易氧化}$$
氧化程度取決於醇的級數
溫度決定產物
濃硫酸 + 乙醇：140°C → 醚（分子間脫水）、180°C → 烯（分子內脫水）。口訣：「一四醚、一八烯」。

醚（Ether）

通式 $\text{R-O-R'}$。兩個烴基接在同一個氧上。

  • 不含氫鍵的給體，沸點比同碳數醇低很多
  • 化學性質穩定，常作有機溶劑
  • 乙醚（$\text{C}_2\text{H}_5\text{OC}_2\text{H}_5$）：早期麻醉劑，易燃，蒸氣密度大於空氣

酚（Phenol）

OH 直接接在苯環上：$\text{C}_6\text{H}_5\text{OH}$。

  • 弱酸性：可與 NaOH 反應（醇不行）
  • 遇 $\text{FeCl}_3$ 呈紫色（鑑別反應）
  • 有殺菌力（石碳酸），用於消毒
$$\text{C}_6\text{H}_5\text{OH} + \text{NaOH} \rightarrow \text{C}_6\text{H}_5\text{ONa} + \text{H}_2\text{O}$$
酚 + 強鹼 → 酚鈉 + 水（醇無此反應）

醇 vs 酚 比較

醇酚
OH 位置接在飽和碳接在苯環碳
酸性中性弱酸性
與 NaOH不反應反應
FeCl₃ 呈色無紫色
與 Na反應放 H₂反應放 H₂

1-8 醛與酮

兩者都含有羰基（$\text{C}{=}\text{O}$），差別在於羰基碳上的取代基。

醛（Aldehyde）

通式 $\text{RCHO}$

羰基碳至少接一個 H

可被氧化為羧酸

酮（Ketone）

通式 $\text{RCOR'}$

羰基碳兩側都是烴基

不易被氧化

常見醛、酮

名稱分子式特性 / 用途
甲醛$\text{HCHO}$37% 水溶液 = 福馬林（防腐）、致癌；合成酚醛樹脂
乙醛$\text{CH}_3\text{CHO}$酒精代謝中間物、合成醋酸原料
丙酮$\text{CH}_3\text{COCH}_3$最簡單的酮、去光水、有機溶劑

醛的鑑別反應

多倫試驗（銀鏡反應）：

$$\text{RCHO} + 2[\text{Ag(NH}_3)_2]^+ + 2\text{OH}^- \rightarrow \text{RCOO}^- + 2\text{Ag}\downarrow + 3\text{NH}_3 + \text{H}_2\text{O}$$
醛被氧化為羧酸根，Ag⁺ 被還原為銀鏡

斐林試驗：

$$\text{RCHO} + 2\text{Cu}^{2+} + 5\text{OH}^- \rightarrow \text{RCOO}^- + \text{Cu}_2\text{O}\downarrow\text{（磚紅色）} + 3\text{H}_2\text{O}$$
醛使 Cu²⁺ 還原為 Cu₂O 磚紅色沉澱
醛 vs 酮 鑑別
醛可進行銀鏡反應和斐林試驗（因為可被氧化）；酮不行。這是最重要的鑑別方式。

1-9 羧酸與酯

羧酸（Carboxylic Acid）

通式 $\text{R-COOH}$。含有羧基（$\text{-COOH}$），為有機酸。

名稱分子式特性
甲酸（蟻酸）$\text{HCOOH}$蟻蟲分泌、有刺激性、兼具醛基可銀鏡反應
乙酸（醋酸）$\text{CH}_3\text{COOH}$食醋成分（3-5%）、冰醋酸（純，mp 16.6°C）
草酸$\text{(COOH)}_2$二元酸、除鏽、有毒
苯甲酸$\text{C}_6\text{H}_5\text{COOH}$食品防腐劑

酸性比較：

$$\text{強無機酸} > \text{羧酸（弱酸）} > \text{碳酸} > \text{酚} > \text{醇（中性）}$$
酸性由強到弱排列
甲酸的特殊性
甲酸結構中同時含有醛基和羧基（$\text{H-COOH}$），因此既有羧酸性質，也能進行銀鏡反應和斐林試驗。

酯（Ester）

羧酸 + 醇 → 酯 + 水（酯化反應）：

$$\text{R-COOH} + \text{R'-OH} \xrightleftharpoons[\text{水解}]{\text{H}^+, \Delta} \text{R-COOR'} + \text{H}_2\text{O}$$
酯化反應為可逆反應，加酸催化
  • 酯類通常有水果香味
  • 不溶於水，密度小於水
  • 沸點比同碳數羧酸、醇低（無法形成氫鍵）

皂化反應（酯的鹼性水解）：

$$\text{油脂} + 3\text{NaOH} \xrightarrow{\Delta} 3\text{肥皂（脂肪酸鈉）} + \text{甘油}$$
油脂 = 高級脂肪酸 + 甘油的酯
常見酯類香味
酯香味
乙酸異戊酯香蕉
丁酸乙酯鳳梨
乙酸辛酯橘子
水楊酸甲酯冬青（撒隆巴斯）
阿斯匹靈
乙醯水楊酸（$\text{aspirin}$）= 水楊酸（酚類）+ 醋酸的酯化產物。具解熱鎮痛作用。

1-10 胺與醯胺

胺（Amine）

可視為氨（$\text{NH}_3$）的氫被烴基取代：

分類通式範例
一級胺（1°）$\text{R-NH}_2$甲胺 $\text{CH}_3\text{NH}_2$
二級胺（2°）$\text{R}_2\text{NH}$二甲胺 $(\text{CH}_3)_2\text{NH}$
三級胺（3°）$\text{R}_3\text{N}$三甲胺 $(\text{CH}_3)_3\text{N}$
  • 胺類為鹼性（氮上有孤對電子，可接受 $\text{H}^+$）
  • 低碳數胺有魚腥味（三甲胺為魚腥味來源）
  • 可與酸反應生成胺鹽：$\text{R-NH}_2 + \text{HCl} \rightarrow \text{R-NH}_3^+\text{Cl}^-$
生物中的胺
胺基酸含有胺基（$\text{-NH}_2$）和羧基（$\text{-COOH}$），是蛋白質的基本單元。生物鹼（如咖啡因、嗎啡、尼古丁）也都含有胺基結構。

醯胺（Amide）

羧酸 + 胺 → 醯胺 + 水（脫水縮合）。含有醯胺鍵 $\text{-CO-NH-}$。

$$\text{R-COOH} + \text{H}_2\text{N-R'} \xrightarrow{\Delta} \text{R-CONH-R'} + \text{H}_2\text{O}$$
醯胺鍵（肽鍵）的形成

重要醯胺：

  • 尿素（$\text{(NH}_2)_2\text{CO}$）：人體蛋白質代謝終產物，化肥原料。歷史意義：1828 年維勒首次從無機物合成有機物（顛覆「生命力」說）
  • 蛋白質中的肽鍵（$\text{-CO-NH-}$）即醯胺鍵
  • 耐綸（Nylon）：己二酸 + 己二胺 → 聚醯胺（合成纖維）
醯胺鍵 = 肽鍵
胺基酸之間透過脫水形成的 $\text{-CO-NH-}$ 鍵就是肽鍵。兩個胺基酸 → 二肽 + 水；多個 → 多肽 → 蛋白質。

總整理：氧化程度與轉化關係

$$\text{烴} \xrightarrow{\text{鹵化}} \text{鹵化烴} \xrightarrow{\text{水解}} \text{醇}$$ $$\text{1° 醇} \xrightarrow{[\text{O}]} \text{醛} \xrightarrow{[\text{O}]} \text{羧酸}$$ $$\text{2° 醇} \xrightarrow{[\text{O}]} \text{酮}$$ $$\text{羧酸} + \text{醇} \xrightarrow{} \text{酯}$$ $$\text{羧酸} + \text{胺} \xrightarrow{} \text{醯胺}$$
有機化合物的氧化層級與互相轉化關係

沸點比較原則

同碳數下：

$$\text{羧酸} > \text{醇} > \text{醛/酮} \approx \text{醚/酯} > \text{烴}$$
  • 羧酸：雙分子氫鍵（二聚體），沸點最高
  • 醇：分子間氫鍵
  • 醛/酮：極性，但無法自身形成氫鍵
  • 醚/酯：極性較弱
  • 烴：僅凡得瓦力
