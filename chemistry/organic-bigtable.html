B. 沸點 / 溶解度比較

同碳數沸點階梯：

$$\text{醯胺} > \text{羧酸} > \text{醇} > \text{胺(1°)} > \text{醛/酮} \approx \text{酯/醚} > \text{鹵化烴} > \text{烴}$$
關鍵：是否能形成氫鍵 + 氫鍵強度 + 雙分子締合（羧酸是二聚體）
類別分子間作用力溶於水原因
醯胺強氫鍵（N-H 與 C=O）低碳數可同時有 H 給體與 C=O 接受體
羧酸雙分子氫鍵（二聚體）C₁～C₄ 任意O-H 同時做給體與接受體
醇氫鍵C₁～C₃ 任意O-H
胺（1°、2°）較弱氫鍵低碳數可N-H（N 電負度小於 O）
胺（3°）無 N-H 氫鍵仍可（孤對接受 H）N 上孤對電子為氫鍵接受體
醛、酮偶極-偶極低碳數可（如丙酮）C=O 為氫鍵接受體
酯偶極-偶極低無 H 給體
醚偶極-偶極（弱）低（醇異構物中最低）無 H 給體
鹵化烴偶極 + 凡得瓦不溶非極性骨架佔多數
烴凡得瓦不溶無極性
同分異構物常考對比
C₂H₆O：乙醇（沸點 78°C，氫鍵）vs 甲醚（−24°C，無氫鍵）。
C₃H₆O：丙醛、丙酮、烯醇/環氧 都可能。
C₂H₄O₂：乙酸（118°C）vs 甲酸甲酯（32°C）。

C. 酸鹼性比較

酸性由強到弱（同碳數可比較）：

$$\text{無機強酸} > \text{羧酸} > \text{H}_2\text{CO}_3 > \text{酚} > \text{H}_2\text{O} \approx \text{醇} > \text{醯胺} > \text{胺}$$

鹼性由強到弱：

$$\text{NaOH} > \text{脂肪族胺}(\text{R-NH}_2) > \text{NH}_3 > \text{芳香胺}(\text{Ar-NH}_2) > \text{醯胺}$$
芳香胺 N 上孤對電子被苯環共振拉走，鹼性減弱；醯胺 N 上孤對被 C=O 拉走，幾乎不顯鹼性
檢驗試劑羧酸酚醇醯胺胺
石蕊試紙變紅幾乎不變不變不變變藍
NaOH(aq)溶解 + 中和溶解（成酚鈉）不反應水解需強加熱不反應
NaHCO₃(aq)冒泡（CO₂）不反應不反應不反應不反應
HCl(aq)不反應不反應不反應水解需強加熱溶解（成胺鹽）
Na 金屬放 H₂放 H₂放 H₂不反應1°、2° 可放
羧酸 vs 酚：用 NaHCO₃ 區分
兩者都能溶於 NaOH，但只有羧酸強到能與 $\text{NaHCO}_3$ 反應冒 $\text{CO}_2$（因為羧酸酸性 > 碳酸）。酚酸性 < 碳酸，不會冒泡。

D. 氧化還原與互換鏈

整個 1-5～1-10 的核心地圖：碳氧化態由低到高，每個官能基代表一個層級。

$$\text{烷} \xrightarrow[\text{光/熱}]{X_2} \text{鹵化烴} \xrightarrow{\text{OH}^-} \text{醇}$$ $$\text{1° 醇} \xrightarrow[\text{部分}]{[\text{O}]} \text{醛} \xrightarrow{[\text{O}]} \text{羧酸}$$ $$\text{2° 醇} \xrightarrow{[\text{O}]} \text{酮}$$ $$\text{3° 醇} \xrightarrow{[\text{O}]} \text{(不易氧化)}$$ $$\text{羧酸} + \text{醇} \xrightleftharpoons[\text{H}_2\text{O}]{\text{H}^+} \text{酯}$$ $$\text{羧酸} + \text{胺} \xrightarrow{-\text{H}_2\text{O}} \text{醯胺（肽鍵）}$$ $$\text{烯} \xleftarrow[\text{180°C}]{\text{濃 H}_2\text{SO}_4} \text{醇} \xrightarrow[\text{140°C}]{\text{濃 H}_2\text{SO}_4} \text{醚}$$
碳氧化態−3 ~ −2−1+1+2+3
代表烷 / 烯 / 炔醇、鹵化烴醛酮羧酸 / 酯 / 醯胺
方向←——— 還原（加 H、減 O） ｜ 氧化（加 O、減 H）———→
「一四醚、一八烯」
乙醇 + 濃硫酸：140°C 分子間脫水生醚；180°C 分子內脫水生烯。同樣是脫水，溫度決定哪一個 OH 被搶。

3. 消去反應（Elimination）

  • 醇脫水（180°C 濃 H₂SO₄）→ 烯
  • 鹵化烴脫 HX（KOH/醇溶液）→ 烯

4. 氧化反應

  • 1° 醇 → 醛 → 羧酸（$\text{KMnO}_4$ / $\text{K}_2\text{Cr}_2\text{O}_7$）
  • 2° 醇 → 酮
  • 3° 醇、酮、醚：不易被一般氧化劑氧化
  • 醛 → 羧酸（極易，連空氣都行）

5. 還原反應

  • 醛 → 1° 醇；酮 → 2° 醇（H₂/Ni 或 NaBH₄）
  • 羧酸、酯 → 1° 醇（LiAlH₄）
  • 硝基苯 → 苯胺（Sn / HCl）

6. 縮合 / 水解

正向（縮合）逆向（水解）
羧酸 + 醇 → 酯 + H₂O（H⁺ 催化）酯 + H₂O / H⁺ → 羧酸 + 醇
羧酸 + 胺 → 醯胺 + H₂O（加熱）醯胺 + H₂O / H⁺ 或 OH⁻ → 羧酸 + 胺
胺基酸 ⇌ 二肽 ⇌ 多肽（蛋白質）蛋白質 + 酶 → 胺基酸（消化）

H. 易混淆陷阱

必背陷阱清單

  1. 甲酸是雙面人：$\text{H-COOH}$ 同時有醛基（H-C=O）和羧基（-COOH），能銀鏡、能斐林，且為羧酸中酸性最強。
  2. 醯胺幾乎中性，不像胺：N 上孤對電子被 C=O 共振拉走，鹼性消失。
  3. 苯胺鹼性弱於氨：苯環共振拉走孤對；脂肪胺則因 R 推電子，鹼性比氨強。
  4. 酚 ≠ 醇：酚是 OH 直接接苯環碳；苯甲醇（$\text{C}_6\text{H}_5\text{CH}_2\text{OH}$）OH 接 sp³ 碳，仍是醇。
  5. FeCl₃ 紫色僅鑑別酚，不能用來抓羧酸或醇。
  6. 斐林對芳香醛弱反應：苯甲醛在斐林中幾乎不變色，銀鏡仍可正向。
  7. 3° 醇不易氧化，但可被脫水成烯。氧化、脫水兩件事不要搞混。
  8. 酯的水解兩種環境：酸催化可逆；NaOH 鹼水解（皂化）為不可逆，因為產物羧酸根進一步離開平衡。
  9. 「酸 + 醇 → 酯」要加 H⁺ 催化、加熱，且須除水才能推動平衡。
  10. 耐綸 vs 蛋白質：兩者主鏈都靠醯胺鍵連結；蛋白質單體是胺基酸，耐綸是己二酸 + 己二胺。
  11. 苯的反應以取代為主，不是加成。題目給「+ Br₂ 取代」要記得需 FeBr₃；無催化劑就不反應。
  12. 乙醚 ≠ 乙酸，乙醚 ≠ 醇：$\text{C}_2\text{H}_5\text{OC}_2\text{H}_5$ 是醚（C-O-C），名字裡的「醚」不要看成「醇」。

三句話總結

  • 看結構辨類別：先找 C=O 的左右鄰居，再找 O-H、N、X。
  • 看反應辨類別：銀鏡 / 斐林 → 醛、甲酸；NaHCO₃ 冒泡 → 羧酸；FeCl₃ 紫 → 酚；HCl 溶解 → 胺。
  • 看物性辨類別：氫鍵強度＝沸點高低；能否做 H 給體決定能否自身氫鍵。
