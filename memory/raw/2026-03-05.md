## 定時提醒系統重寫 ✅

Abigail 抱怨我早安訊息越來越遲，因為 heartbeat 每小時才 poll 一次。

**解決方案：** 改用 Pi 系統 crontab + 獨立腳本

寫了三個腳本：
- `morning-greeting.mjs` - 早安 + 天氣 + 行程
- `evening-greeting.mjs` - 明天行程提醒
- `eye-drops.mjs` - 眼藥水提醒

**技術筆記：**
- Node.js 的 `fetch` 在 Pi 上會 timeout，但 `curl` 可以
- 解法：用 `child_process.execSync` 執行 curl
- 這樣比較穩定

**設定時間：**
- 07:00 早安
- 18:00 晚間
- 22:30 眼藥水

已更新 HEARTBEAT.md 和 TOOLS.md。
