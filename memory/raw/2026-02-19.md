# 2026-02-19

## MONO 網站大更新日 (v1.2.0 → v1.4.4)

### 計時器持久化 (v1.2.x)
- 實作計時器狀態持久化到 localStorage
- 修復 iOS 背景時 JS 暫停問題 - 用 `visibilitychange` 事件重新計算時間
- 清快取不再清除計時器狀態

### 主題同步修復 (v1.2.3 - v1.2.6)
- 問題：部分講義頁面不跟著主題變色
- 原因：CSS 有 hardcoded `:root` 顏色值
- 解法：移除各頁面的 `:root` block，theme.js 移到 `<head>` 提早執行
- theme.js 加入 IIFE 立即套用主題

### 樣式改進
- 螢光筆 `<mark>` 加圓角 (border-radius: 4px)
- 螢光筆文字顏色跟主題走 (var(--text))
- 筆記框左邊線改用 var(--accent)
- 移除各頁面 hardcoded 的 #d4a000 黃色

### 首頁 Logo 重新設計 (v1.3.x)
- Abigail 用 Canva 設計了新 logo：圓圈+中心點+MONO 文字
- 我把 #000000 換成 currentColor 讓它跟主題走
- Logo 縮小到 70px，上方 padding 減少

### 統計面板大改版 (v1.4.0+)
- 建立獨立的 `focus-stats.html` 統計頁面
- 功能：
  - 今日/本週/本月 tabs 切換
  - 總時數卡片 + 與上期比較百分比
  - 平均專注時長、專注次數、連續天數
  - 最常專注時段（早/午/晚）
  - 最常讀的科目
  - 長條圖（日檢視=每小時、週檢視=每天）
  - 甜甜圈圖 - 科目分布
  - 熱力圖日曆（月檢視）
  - 科目篩選
  - 滑動/點擊刪除記錄
- 長條圖可左右滑動
- 修復 bar overflow 問題（進行中）

### 技術筆記
- PWA 快取很頑固，要清除需要：清 Service Worker + caches API
- 外部 SVG 用 `<img>` 無法繼承 CSS 顏色，要用 inline SVG + currentColor
- iOS Safari 背景 JS 約 5-10 秒後暫停，回來時用 visibilitychange 重算
