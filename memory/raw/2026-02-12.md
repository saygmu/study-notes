## Cron 提醒機制 - 終於解決！

**問題：** `deliver: true` 的方式不可靠，訊息常常送不到

**解法：** 讓 isolated session 的 agent 直接用 message tool 發送

```json
{
  "sessionTarget": "isolated",
  "payload": {
    "kind": "agentTurn",
    "message": "用 message tool 發送以下訊息到 Telegram (channel: telegram, to: 744312344)：\n\n[提醒內容]\n\n發送完畢後結束。"
  }
}
```
